# Rust Programming

Welcome to the Rust Programming project!

## Getting Started

1. Install [Rust](https://www.rust-lang.org/tools/install).
2. Clone this repository.
3. Build and run the project:

```sh
cargo build
cargo run
```

## Project Structure

- `src/` - Source code directory
- `Cargo.toml` - Project configuration
